# GitHub Actions Secrets

Go to: **Settings → Secrets and variables → Actions → New repository secret**

| Secret name | Description |
|---|---|
| `STAGING_DB_PASSWORD` | PostgreSQL password |
| `STAGING_KEYCLOAK_ADMIN_PASSWORD` | Keycloak admin password |
| `STAGING_KC_DB_PASSWORD` | Keycloak Postgres password |
| `STAGING_KC_BACKEND_CLIENT_SECRET` | Keycloak backend client secret |
| `STAGING_KC_USER_ADMIN_PASSWORD` | Test user inv_admin password |
| `STAGING_KC_USER_CLERK_PASSWORD` | Test user inv_clerk password |
| `STAGING_KC_USER_AUDITOR_PASSWORD` | Test user inv_auditor password |
| `STAGING_KC_USER_VIEWER_PASSWORD` | Test user inv_viewer password |
| `STAGING_GRAFANA_ADMIN_PASSWORD` | Grafana admin password |
